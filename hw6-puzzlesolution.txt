HOW I SOLVED THE PUZZLE
ALL THE WORK AND ITERATIONS I WENT THROUGH ARE MORE THOROUGHLY DOCUMENTED IN WORK.IPYNB
STEP 1: TRIED TO DO PUZZLE EASY ON MY OWN SO THAT I COULD REUSE CODE THAT WORKS WITHOUT WORKING ON BUGS WITH THE HARD PUZZLE

STEP 1.A: CODE TO BREAK THE KEY

- ARBITRARILY CHOSE A WORD THAT IS PROBABLY DEFINITELY GOING TO BE IN THE TEXT ("the")
- ITERATING THROUGH NUMBERS FROM 0-9999 TO FIND THE KEY
- FOR EACH KEY GENERATED - A HASH OF KEY + "THE" is created
- THE HASH GENERATED IS THEN CHECKED TO SEE IF IT IS IN PUZZLE_EASY.TEXT
- THIS TOOK 30 SECONDS

ONCE A MATCH IS FOUND THE KEY IS REGISTERED AS THE KEY

STEP 1.B: CREATING A LOOKUP DICTIONARY FOR COMMONLY SEEN WORDS

- ONCE I HAVE THE KEY , I USED A DICTIONARY OF COMMON WORDS TO GENERATE HASHES OF THE WORDS + KEY 
- I COMPARED THIS DICTIONARY TO THE PUZZLE_EASY.TXT FILE AND FOUND THE WORDS I COULD

STEP 1.C: MODIFYING EACH WORD WITH PUNCTUATION ETC
- SINCE I DIDN'T GET MANY WORDS, I REPEATED THE SAME THING AS 1.B WITH A NEW FUNCTION APPLIED TO EACH WORD TO CREATE VARIANTS OF THE WORD.
- THEN I FOUND A BETTER VERSION OF THE TEXT WHICH I THEN MATCHED USING THE INTERNET

STEP 1.D: FINDING THE MISSPELLING
- ONCE I FOUND THE WORD THAT IS MISPELLED , I CREATED AN EDIT DISTANCE ONE FUNCTION
- THE EDIT DISTANCE ONE FUNCTION WAS APPLIED TO THE MISPELLED WORD AND THEN HASHES OF THE FUNC OUTPUT + KEY WERE MATCHED
- THESE WERE MATCHED AGAINST THE UNKNOWN HASH
- IF THAT DIDN'T WORK, I APPLIED THE FUNCTION TO ALL THE OUTPUTS OF THE FIRST TRY TO FIND ALL THE VARIANTS OF EDIT DISTANCE 2
- I THEN REPEATED THE SEARCH
- THAT FOUND GAQDENS

STEP 2: BREAKING THE ACTUAL PUZZLE
STEP 2.A: BREAKING THE KEY
- I USED THE SAME CODE AS FOR THE EASY PUZZLE BUT CROSSCHECKED WITH BOTH "THE" AND "AND" TO PREVENT COLLISIONS AND INCREASED THE SEARCH SPACE TO A BILLION
- THIS TOOK 14 MINUTES TO RUN

STEP 2.B: CREATING THE LOOKUP DICTIONARY
- THIS TIME I USED A PYTHON PACKAGE WITH "FREQUENT WORDS" AND TOOK THE TOP 50000 TO MAKE A BIGGER DICTIONARY

STEP 2.C: FINDING MISPELLED WORD
- I FOUND THE TEXT IN A GAME THEORY TEXT BOOK USING THE INTERNET
- THE MISPELLED WORD WAS CIRCUMSTANCES
- I CREATED EDIT DISTANCE 1 AND EDIT DISTANCE 2 USING THE SAME FORMAT AS 1.D
- I THEN CROSSCHECKED WITH THE UNKNOWN HASH AND FOUND THE ERROR
